Консольное приложение выводит список галактик с их названиями, расстоянием (в мегасветовых годах) и морфологическим типом. В исходном коде содержались два бага.

### Обнаруженные и исправленные ошибки

**Баг №1** — неверный тип свойства:
```
было object надо GType 
```csharp

**Баг №2** — опечатка в символе `case`:
```csharp
было case 'l':  надо case 'I': 
```


### Использованные способы отладки

| Инструмент | Описание применения |
|---|---|
| **Breakpoints** | Поставлена на строку `Console.WriteLine` внутри `foreach` и на конструктор `GType`. |
| **Data Tips** | Наведение на `theGalaxy.GalaxyType` → раскрытие → обнаружение свойства `MyGType`. |
| **Step Into** | Заход в конструктор `GType` для проверки ветки `switch`. |
| **Edit and Continue** | Исправление вызова `GalaxyType.MyGType` прямо во время сеанса отладки. |
| **Restart** | Перезапуск приложения с отладчиком после правки типа свойства. |

## Задание 3 — Приложение «Буквы»


### Результат
<img width="550" height="414" alt="image" src="https://github.com/user-attachments/assets/e527b995-2cc4-4f7d-8409-e27f865d7f3a" />


### Описание

Консольное приложение итерирует по массиву символов `letters`, на каждом шаге строит строку `name` и вызывает `SendMessage`. Является обучающим примером для знакомства с основными инструментами отладчика.

### Использованные способы отладки

| Инструмент | Описание применения |
|---|---|
| **Breakpoints** | Поставлена на строку `name += letters[i]` внутри цикла `for`. |
| **Data Tips** | Раскрытие массива `letters[10]` → просмотр всех элементов; отслеживание `name` на каждой итерации. |
| **Run to Click** | Быстрый переход к строке `Console.WriteLine` внутри `SendMessage`. |
| **Autos** | Автоматическое отображение переменных текущей и предыдущей строк. |
| **Locals** | Просмотр `letters`, `name`, `a`, `i` в текущей области видимости. |
| **Watch** | Добавлена переменная `name` → наблюдение за её ростом по итерациям. |
| **Call Stack** | Просмотр цепочки `Main → RunLetters → SendMessage`; двойной щелчок для навигации. |
| **Restart** | Повторный запуск с начала для проверки поведения при разных точках останова. |
